# 快速上手

### 安装

```bash
# 通过 npm
npm i cbc-ui

# 通过 yarn
yarn add cbc-ui
```
### Props

| 参数          | 说明     | 类型     | 默认值    |
| ------------- | -------- | -------- | --------- |
| applicationName | app的name | _string_ | `-` (设备维保、监控安灯、资产效能、仓储管理、知识库)  
| runningEnv| 运行环境 | _string_ |`h5`(h5、dd、wx)|
